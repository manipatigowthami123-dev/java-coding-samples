# JavaPrograms

This repository contains simple Java programs for beginners, including:

1. *Number Guessing Game*  
   - A program where the user tries to guess a randomly generated number between 1 and 100.  
   - Provides feedback if the guess is too high or too low.  

2. *Palindrome Checker*  
   - A program that checks whether the given word or number is a palindrome.  
   - Takes dynamic input from the user.
